# 陪伴产品

AI 情感陪伴产品项目仓库，包含产品调研、PRD 文档和应用代码。

## 项目结构

```
├── niandoubao/                          # 粘豆包 - Next.js Web App
├── vibe-prd-情绪疗愈/                    # 产品需求文档 & 风格预览
└── product-insight-report-*.html         # 产品洞察调研报告
```

## 粘豆包（NianDouBao）

面向 18-28 岁年轻女性的 AI 情绪陪伴工具，核心功能为"每日情绪签到 + AI 引导 + 微行动闭环"。

- **技术栈**：Next.js 14 / TypeScript / Tailwind CSS / Supabase / Kimi API
- **启动方式**：`cd niandoubao && npm install && npm run dev`
